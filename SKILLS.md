# 프로젝트 개발 환경
1. 프로젝트 기본 정보
 * 명칭: 풀사이클 생성형 AI OTT 미디어 서비스 팀 프로젝트
 * 팀이름: VODA
 * OTT 주 서비스: 인기 영화 큐레이션 OTT
 * 기간: 2026.03.24(화) ~ 2026.04.03(금) (주말 제외 총 9일)
 * 배포 구조: 모노레포 (frontend/ + backend/), Render 서비스 활용

2. 팀 구성 및 역할
 * 인원: 총 4인
     * A,B (디자인 리드): 디자인 시스템 및 시각 요소 담당
     * C (프론트엔드 리드): 클라이언트 구현 담당
     * D (백엔드/AI 리드): 서버 및 AI 모델 통합 담당
 * 숙련도 특징: 디자인(80%) 비중이 높고 코딩(20%) 비중이 낮음 → Gemini CLI 바이브 코딩으로 구현 보완

3. 기술 스택 (Tech Stack)
 * 기획/디자인: Google Stitch (아이데이션), Figma (디자인 시스템)
 * 프론트엔드:
     * React 19, React Router v7
     * Tailwind CSS v4, tailwind-merge
     * react-player (미디어 재생)
     * Axios (HTTP 통신), FontAwesome (아이콘)
 * 백엔드: Python FastAPI
 * AI 및 데이터:
     * TMDB API (영화/미디어 데이터)

4. 프로젝트 수행 단계
 1. 기획 (2일): 서비스 방향성 및 요구사항 정의
 2. 디자인 (3일): UI/UX 및 디자인 시스템 구축
 3. 구현 (3일): 프론트엔드, 백엔드 및 AI 기능 개발
 4. 검토 및 배포 (1일): 최종 테스트 및 운영 환경 배포
